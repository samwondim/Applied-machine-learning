## Getting started

### starting the python backend

* ```bash

python3 -m venv .env

```

* ```bash
source .env/bin/activate
```

* ```bash

pip install -r requirements.txt

```

* ```bash
python3 app.py
```

### starting the web frontend

* ```bash

cd front

```
* ```bash
npm install
```

* ```bash

npm run dev

```
